# 🏎️ Retro 3D Racer 

A modern, 3D reimagining of the classic Java button-phone racers of the past, brought to life with **AI-powered motion control.** 

This project is a tribute to the nostalgic endless racers we grew up with, now upgraded for the browser and controlled entirely by your own movements.

---

## 🎮 How to Play

**[👉 Play the Game Here!](https://abrahammanjolo.github.io/retro-car-racer-game/)**

*   **Steering:** Move your head left or right in front of your webcam to steer.
*   **Jump:** Press the **[J]** key to leap over obstacles.
*   **Pause:** Press the **[SPACE]** key to pause/resume.
*   **Goal:** Survive as long as possible. The game gets faster and more intense the longer you stay on the road!

---

## 🚀 Features

*   **Nostalgic Vibe:** Inspired by the simple, high-speed fun of classic mobile Java games.
*   **FACIAL RECOGNITION Steering:** Uses **MediaPipe Face Mesh** to track your head position via webcam, making the game feel responsive and immersive.
*   **3D Engine:** Built from scratch using **Three.js** to deliver a smooth, retro-styled 3D experience.
*   **Dynamic Audio:** Features a real-time synth engine that mimics the roar of a sports car.

---

## 🛠️ Built With

*   **[Three.js](https://threejs.org/):** For 3D rendering and scene management.
*   **[MediaPipe](https://developers.google.com/mediapipe):** For real-time facial landmark detection.
*   **Web Audio API:** For custom engine sound synthesis.
*   **GitHub Pages:** For hosting and deployment.

---

## 💡 About the Project
I built this project to bridge the gap between retro gaming history and modern web AI capabilities. By moving away from traditional button inputs and using facial tracking, I wanted to turn a casual racer into an interactive, physical experience.
